# Healthcare Decision Support System - CrewAI

AI-powered healthcare assistant with session memory and multi-agent collaboration.

## Features

- Multi-Agent CrewAI Architecture
- Session Memory (remembers your name)
- Hospital Data Analysis (40+ metrics)
- Distance Calculations
- Conversational AI

## Quick Start

```bash
# Install
pip install -r requirements.txt

# Configure .env
OPENAI_API_KEY=your_key_here

# Run
python main.py
```

Open http://localhost:8001

## Example Queries

- "My name is John"
- "How many hospitals?"
- "Show me beds at City General Hospital"
- "Calculate distance between hospitals"

## Tech Stack

- CrewAI + OpenAI GPT-4
- FastAPI
- Python 3.10+
